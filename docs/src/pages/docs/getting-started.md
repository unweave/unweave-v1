You're ready to train your model. There's only one piece missing, the GPU to train on. With Unweave, you're 2 mins away to start training your model. 

## Install the cli
```bash
$ brew tap unweave/unweave
$ brew install unweave
```

## Login
```bash
$ unweave login
```

## Create a machine
```bash
unweave new
```

## Launch vscode

```bash
unweave code
```
